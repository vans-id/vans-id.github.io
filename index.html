Project : AvangeGram

HTML & CSS - Javascript - PHP - MySQL - Laravel
- VueJs - Bootstrap

Made by Laravel for the backend and some of Vue js and Bootstrap on the front end. This app is a clone of the real instagram, which includes : Login and logout, Signup as a new user, Add a new post, Edit our Bio, See detail of a post, Search another user, then follow them, Protected Routes, Feed page, which includes posts from other people we follow

HTML & CSS - Bootstrap - Javascript - jQuery - ajax - public API

Using oMDB public API, this app serve movies which we search the title for. The app is made of from a lot Javascript and jQuery to make ajax request works. By clicking details link, it will show up a modal box which shows the detail of each movie.
